## [0.1.0] - 2026-05-05

### Added
- Initial release
- Extract agent conversations from OpenCode SQLite database
- Export sessions to JSON, Markdown, and HTML formats
- Hash-based file collision detection
- CLI interface with argparse
- List sessions with optional search and limit
